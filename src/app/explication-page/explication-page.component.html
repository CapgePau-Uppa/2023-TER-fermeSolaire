About

What is this projet ?

This website is a school project done by two students, MANUEL Anthony and STEMMEL Nathanael, in first year of UPPA's Master degree TI. It has been supervised by LAMAS Fabien, ANSELMI Thomas, KOLLEN Thibault, FRANZESE Ricardo, FAUVET Guillaume, DUBOIS Nicolas and LEFEVRE Mikaël , who all work for Capgemini.

The goal of it is to show thanks to a heatmap the best places to put solar panels. It currently only shows data for mainland France.

Technologies used

The web site's architecture is seperated in two.

The server side, which treats data and answers requests (it's made using NodeJS);

and the client side, which requests data to the server (it was made using Angular).
